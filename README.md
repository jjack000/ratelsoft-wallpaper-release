# RatelSoft WallPaper — Releases

이 저장소는 **배포(Release) 산출물만** 보관하는 공개 저장소입니다.
- 소스코드는 포함하지 않습니다.

## 다운로드
1. 상단의 **Releases** 탭에서 최신 릴리즈를 선택합니다.
2. Assets에서 필요한 파일(설치/압축/실행 파일 등)을 다운로드합니다.

> 어떤 파일을 받아야 할지 애매하면, 릴리즈 노트(Release notes)에 안내된 “권장 다운로드”를 우선 따르세요.

## 폴더/파일 구성(예시)
릴리즈마다 구성은 조금 다를 수 있습니다.
- `WallPaper-...` : WallPaper 본 프로그램 배포물
- `CameraTestApp-...` : 카메라 테스트용 배포물
- `*.zip` / `*.7z` : 압축 배포 패키지
- `*.exe` : 실행 파일(또는 설치 파일)
- `*.json` / `*.yml` : 릴리즈 메타데이터/설정 샘플(있는 경우)

## 설치 및 실행
- 기본적으로 Windows 환경을 대상으로 합니다.
- 실행 전 필수 구성(드라이버/장치 연결/설정 파일 등)은 각 릴리즈 노트에 명시됩니다.

## 변경 내역(Changelog)
변경 내역은 각 릴리즈의 **Release notes**에 요약되어 있습니다.

## 이슈(버그/요청)
이슈는 이 저장소에서 활성화되어 있으며, 아래 정보를 함께 남겨주시면 재현/수정에 도움이 됩니다.
- 사용 중인 릴리즈 버전(태그)
- OS 버전
- 재현 절차(가능하면 최소 단계)
- 기대 동작 / 실제 동작
- 로그/스크린샷(개인정보/민감정보는 제거 후 첨부)

## 보안 관련 제보
보안 취약점/민감 이슈는 공개 이슈 대신 별도 채널로 제보해 주세요(사내 절차 또는 담당자에게 직접 전달).
